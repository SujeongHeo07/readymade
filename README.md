# readymade

작가들이 만들어놓은 레디메이드 표지를 한 곳에 모아 비교해서 구입할 수 있는 사이트로 기획부터 퍼블리싱까지 함께 만든 2인 팀프로젝트 입니다.

📖 프로젝트 정보

- 프로젝트명: READY:MADE
- 구분: 팀프로젝트
- 기여도: 기획 50%, 디자인 0% , 퍼블리싱 100%
- 작업기간: 2023.05.14 ~ 2023.06.10
- 사용스킬: HTML5, CSS3, jQuery, JS, Figma
- 디바이스: Pc
- 라이브러리: Swiper, CKEditor5
